# valiant

`valiant` is a 2D game engine built with SDL2.

## Getting Started

### Dependencies

Install [SDL2](https://libsdl.org/download-2.0.php).

Testing requires [Catch2](https://github.com/catchorg/Catch2/).

## Testing

Build and run tests:

```shell
$ git clone https://github.com/claby2/valiant.git
$ cd valiant
$ cmake .
$ make test
$ ./test
```
